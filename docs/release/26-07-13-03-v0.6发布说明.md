# AIpane v0.6 Release Notes

**Date**: 2026-07-13
**Tag**: `v0.6`
**Commit**: `83e320b`

---

## What's New

### AI Service Configuration Management

Add, edit, remove, and reorder custom AI services directly from the settings panel. Supports up to 100 user-defined services with name and URL validation. Built-in services (DeepSeek, ChatGPT, Claude, etc.) remain available alongside your custom entries.

### Startup Performance Optimization

Significant reduction in cold-start latency:

- **Native JSON config** — Replaced `electron-store` and `uuid` dependencies with native Node.js `fs` + `crypto.randomUUID()`, cutting ~2 MB from the dependency tree and eliminating two `require()` calls on the critical path.
- **Early window display** — Window now appears at `ready-to-show` instead of waiting for the full page to finish loading, making the first frame visible earlier.
- **Seed injection** — Initial settings are injected synchronously via `window.__INITIAL_SETTINGS__`, eliminating the first `settings:get` IPC round-trip and preventing the "empty panel flash" on startup.
- **Lazy SettingsPanel** — SettingsPanel is now lazily mounted (only loaded on first click) and code-split into its own chunk via `React.lazy`, reducing the main bundle's first-render work.

## Bug Fixes

- Fixed the settings gear button overlapping the top-right panel in horizontal layout mode.
- Fixed macOS window not showing on launch and white screen on reopen after closing all windows.
