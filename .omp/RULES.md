# RULES

## 双语文档约定
- README 双语文档：`README.md`（英文）为主，`README_zh.md`（中文）保持同步，每次 git 提交前需更新两份文档
- 项目文档内容：内部文档（docs/plan/、docs/report/ 等）使用中文撰写

## Git 提交
- 格式：`type: [YYYY-MM-DD] 简短描述`，其中 type 为 `feat`/`fix`/`docs`/`chore`/`refactor`/`test` 等
- 描述使用英文（English）

## 文档命名
- docs/ 下的文件命名格式：`YY-MM-DD-0X-描述.md`，其中 `0X` 为当天第 X 份文档（01 起），描述用中文
